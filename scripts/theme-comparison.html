How to Add New Themes

1. Add Colors to Tailwind Config

// Add to tailwind.config.js
colors: {
    yourTheme: {
        50: '#f0f9ff',   // Lightest
        100: '#e0f2fe',
        200: '#bae6fd',
        300: '#7dd3fc',
        400: '#38bdf8',
        500: '#0ea5e9',  // Primary
        600: '#0284c7',
        700: '#0369a1',
        800: '#075985',
        900: '#0c4a6e',  // Darkest
    }
}

2. Copy Theme Card Template

<div class="theme-card bg-white rounded-2xl p-6 shadow-lg">
    <h3 class="text-xl font-bold text-gray-900 mb-4">Your Theme Name</h3>
    <div class="space-y-4">
        <!-- Success Screen Preview -->
        <div class="bg-gradient-to-br from-yourTheme-50 via-secondary-50 to-accent-50 border border-yourTheme-200 rounded-xl p-4">
            <!-- Content here -->
        </div>
        <!-- Color Palette -->
        <!-- Theme Info -->
    </div>
</div>
